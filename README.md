🧠 AI Image Caption & Hashtag Generator

An AI-powered web application that generates creative captions and relevant hashtags from uploaded images.

🚀 Features

Upload an image

AI-generated caption

Auto-generated relevant hashtags

Dynamic rotating loading messages

Clean responsive UI

FastAPI backend with Python

Git version controlled

🛠 Tech Stack

Frontend:

HTML

CSS

JavaScript

Backend:

Python

FastAPI

Uvicorn

Deployment:

Render (in progress)

⚙️ How It Works

User uploads an image

Image is sent to FastAPI backend

Python processes the image using AI logic

Caption + hashtags are generated

Loading animation is displayed during processing

Final result is shown to the user

📦 Installation (Run Locally)
git clone <your-repo-link>
cd <project-folder>
pip install -r requirements.txt
uvicorn main:app --reload

Open in browser:

http://127.0.0.1:8000
